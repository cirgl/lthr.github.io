Some useful VIM commands.


```
CTRL + Z            Suspend VIM process (get back to shell)
fg                  .. resume VIM from shell
:sh                 Suspend VIM process (get back to shell)
exit                .. resume VIM from shell

i                   Insert mode
ESC                 Exit insert mode
dd                  Delete line
yy                  Copy line
P                   Paste above/before
r                   Replace character
x                   Delete character
u                   Undo
:q                  Quit
:q!                 Force quit
ZZ                  Save and quit
```
